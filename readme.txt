Instruções para execução:
Bibliotecas necessárias (além das Bibliotecas padrão utilizadas na matéria, como glut e freeglut):
-Soil

Compilação: no linux, execute o comando "make compilar". No Windows, utilize make compilar_windows
Execução: execute o comando "make executar"

Navegação: Use WASD para movimentar pelo cenário.
Use as teclas 1-4 para apagar ou acender as luzes de um cômodo.
Use a tecla 5 para apagar ou acender a luz do cenário (Sol)
Use as teclas + e - para ajustar a intensidade da luz do cenário
Use as teclas o e p para ajustar a neblina (fog) do cenário
